# –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∞—è —Å–µ–∫—Ü–∏—è
## –ü—Ä–µ–¥–ª–æ–∂–µ–Ω–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ
–û—á–µ–Ω—å –Ω–∞–¥–µ—é—Å—å, —á—Ç–æ –æ–±—Ä–∞—Ç–Ω–∞—è —Å–≤—è–∑—å –Ω–∞ –æ–±—Å—É–∂–¥–µ–Ω–∏–∏ –∑–∞–¥–∞—á–∏ –±—ã–ª–∞ –∏—Å–∫—Ä–µ–Ω–Ω–µ–π, –ø–æ—Ç–æ–º—É —á—Ç–æ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–Ω—ã–π –º–Ω–æ–π –∏ –æ–∑–≤—É—á–µ–Ω–Ω—ã–π –ø–æ–¥—Ö–æ–¥ c `std::set` —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã –∏, –∫–∞–∫ —Å–ª–µ–¥—Å—Ç–≤–∏–µ, –æ–¥–∏–Ω–∞–∫–æ–≤–æ –Ω–µ–≤–µ—Ä–Ω—ã.

–Ø —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–∏ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–ª –≤—Ä—É—á–Ω—É—é –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—É—é –≤–µ—Ä—Å–∏—é `std::set_intersection`, –∫–æ—Ç–æ—Ä–∞—è –Ω–µ —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏—è, –∞ —Ç–æ–ª—å–∫–æ –ø–æ–¥—Å—á–∏—Ç—ã–≤–∞–µ—Ç —Ä–∞–∑–º–µ—Ä —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞.

–ü—Ä–∏ —ç—Ç–æ–º —è –Ω–µ—è–≤–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª —Å–ª–µ–¥—É—é—â–µ–µ –æ—Ç–Ω–æ—à–µ–Ω–∏–µ –ø–æ—Ä—è–¥–∫–∞:
```c++
bool operator<(const PointType& a, const PointType& b)
{
    return !approx_equal(a, b) && (a < b);
}
```
–≥–¥–µ `approx_equal` ‚Äî —ç—Ç–æ eps-–±–ª–∏–∑–æ—Å—Ç—å –¥–≤—É—Ö —Ç–æ—á–µ–∫ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º max-–º–µ—Ç—Ä–∏–∫–∏:

`std::abs(a.first - b.first) <= eps && std::abs(a.second - b.second) <= eps`

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º `<= eps` –¥–ª—è –ø—Ä–æ—Å—Ç–æ—Ç—ã –¥–∞–ª—å–Ω–µ–π—à–∏—Ö —Ä–∞—Å—Å—É–∂–¥–µ–Ω–∏–π.

–°—Ä–∞–≤–Ω–µ–Ω–∏–µ –≤–µ–ª–∏—á–∏–Ω —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –Ω–µ–∫–æ—Ç–æ—Ä–æ–π –æ–∫—Ä–µ—Å—Ç–Ω–æ—Å—Ç–∏ `eps` –¥–µ–ª–∞–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –∞—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω—ã—Ö –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤ –∏ –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤ –Ω–∞–ø–æ–¥–æ–±–∏–µ `std::set_intersection` **–Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã–º**, —Ç–∞–∫ –∫–∞–∫ –¥–ª—è –Ω–∏—Ö `operator<` –¥–æ–ª–∂–µ–Ω —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—Ç—å –∫—Ä–∏—Ç–µ—Ä–∏—è–º —Å—Ç—Ä–æ–≥–æ–≥–æ —á–∞—Å—Ç–∏—á–Ω–æ–≥–æ –ø–æ—Ä—è–¥–∫–∞ (strict weak ordering):
- –ê–Ω—Ç–∏—Ä–µ—Ñ–ª–µ–∫—Å–∏–≤–Ω–æ—Å—Ç—å: `a < a` –≤—Å–µ–≥–¥–∞ –ª–æ–∂–Ω–æ
- –ê—Å–∏–º–º–µ—Ç—Ä–∏—á–Ω–æ—Å—Ç—å: –µ—Å–ª–∏ `a < b` –∏—Å—Ç–∏–Ω–Ω–æ, —Ç–æ `b < a` –ª–æ–∂–Ω–æ
- –¢—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç—å: –µ—Å–ª–∏ `a < b` –∏ `b < c`, —Ç–æ `a < c`
- –¢—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç—å –Ω–µ—Å—Ä–∞–≤–Ω–∏–º–æ—Å—Ç–∏ (—ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ—Å—Ç–∏): –µ—Å–ª–∏ `a` –∏ `b` –Ω–µ—Å—Ä–∞–≤–Ω–∏–º—ã, –∏ `b` –∏ `c` –Ω–µ—Å—Ä–∞–≤–Ω–∏–º—ã, —Ç–æ `a` –∏ `c` —Ç–∞–∫–∂–µ –Ω–µ—Å—Ä–∞–≤–Ω–∏–º—ã

**–ù–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –≤–µ–ª–∏—á–∏–Ω —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º eps-–æ–∫—Ä–µ—Å—Ç–Ω–æ—Å—Ç–∏ –Ω–∞—Ä—É—à–∞–µ—Ç —Ç—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç—å —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ—Å—Ç–∏**.

–ï—Å–ª–∏ –º—ã –≤–æ–∑—å–º–µ–º –æ–¥–Ω–æ–º–µ—Ä–Ω—ã–π —Å–ª—É—á–∞–π, –∏ —É –Ω–∞—Å –µ—Å—Ç—å 3 —Ç–æ—á–∫–∏ –Ω–∞ –ø—Ä—è–º–æ–π `[0, eps, 2 * eps]`, —Ç–æ —Ç–æ—á–∫–∏ 1-2 —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã, —Ç–æ—á–∫–∏ 2-3 —Ç–∞–∫–∂–µ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã, –Ω–æ —Ç–æ—á–∫–∏ 1-3 –Ω–µ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã, —Ç–æ –µ—Å—Ç—å —Ç—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç—å –Ω–∞—Ä—É—à–µ–Ω–∞.

–ë–æ–ª–µ–µ —Ç–æ–≥–æ, –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å `eps` –≤ –¥–≤—É–º–µ—Ä–Ω–æ–º —Å–ª—É—á–∞–µ —Ç–∞–∫–∂–µ –Ω–∞—Ä—É—à–∞–µ—Ç –∏ –æ–±—ã—á–Ω—É—é —Ç—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç—å:

–¥–ª—è —Ç–æ—á–µ–∫ `[(0, 0), (0, 1), (eps, 0)]`

–ò–∑ —ç—Ç–æ–≥–æ –ø—Ä–∏–º–µ—Ä–∞ –ª–µ–≥–∫–æ –ø–æ–ª—É—á–∏—Ç—å –Ω–∞–±–æ—Ä –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö, –ø—Ä–∏–≤–æ–¥—è—â–∏–π –∫ –Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–π —Ä–∞–±–æ—Ç–µ –∞–ª–≥–æ—Ä–∏—Ç–º–∞:

–ù–æ –µ—Å–ª–∏ –º—ã –ø–æ–ø—ã—Ç–∞–µ–º—Å—è –∑–∞–ø—É—Å—Ç–∏—Ç—å `std::set_intersection` –¥–ª—è –¥–∞–Ω–Ω—ã—Ö –º–∞—Å—Å–∏–≤–æ–≤, —Ç–æ –Ω–∞ –ø–µ—Ä–≤–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ –º—ã –ø—Ä–æ–ø—É—Å—Ç–∏–º —Ç–æ—á–∫—É –∏–∑ `B` –∏–∑-–∑–∞ —Ç–æ–≥–æ, —á—Ç–æ –æ–Ω–∞ –º–µ–Ω—å—à–µ –ø–µ—Ä–≤–æ–π —Ç–æ—á–∫–∏ –∏–∑ `A`, –∏ –ø–æ–ª—É—á–∏–º –ø—É—Å—Ç–æ–µ –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ. –ß—Ç–æ–±—ã –±—ã–ª–æ –ª–µ–≥—á–µ –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞—Ç—å—Å—è –≤ –¥–∞–Ω–Ω–æ–º —Ä–∞—Å—Å—É–∂–¥–µ–Ω–∏–∏, –º–æ–∂–Ω–æ –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å possible implementation –∑–¥–µ—Å—å:

–ú–æ–∂–Ω–æ –ø—ã—Ç–∞—Ç—å—Å—è –∏–∑–º–µ–Ω–∏—Ç—å `operator<`, –Ω–æ –∏–∑-–∑–∞ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–æ–π –Ω–µ—Ç—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ eps-—Å—Ä–∞–≤–Ω–µ–Ω–∏—è –º—ã –≤—Å–µ–≥–¥–∞ –±—É–¥–µ–º –ø–æ–ª—É—á–∞—Ç—å –æ—à–∏–±–∫–∏ –≤ –∫–∞–∫–∏—Ö-–ª–∏–±–æ —Å–ª—É—á–∞—è—Ö.

## –ü–æ–¥—Ö–æ–¥—ã –∫ –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–º—É —Ä–µ—à–µ–Ω–∏—é

### –ù–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –ø–∞—Ä —Å–æ–≤–ø–∞–¥–∞—é—â–∏—Ö —Ç–æ—á–µ–∫
–í –∏—Ç–æ–≥–µ –º–Ω–µ —É–¥–∞–ª–æ—Å—å –Ω–∞–π—Ç–∏ 3 –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ —Ä–∞–±–æ—Ç–∞—é—â–∏—Ö –ø–æ–¥—Ö–æ–¥–∞ –¥–ª—è –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è —Å–æ–≤–ø–∞–¥–∞—é—â–∏—Ö —Ç–æ—á–µ–∫:
1. –û—á–µ–≤–∏–¥–Ω—ã–π bruteforce –∑–∞ *O(N¬≤)*, –Ω–æ —ç—Ç–æ —è–≤–Ω–æ –Ω–µ —Ç–æ, —á—Ç–æ –æ–∂–∏–¥–∞–µ—Ç—Å—è
2. Quad-tree / k-d tree / BSP tree –¥–ª—è –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ —Å —Ü–µ–Ω—Ç—Ä–∞–º–∏ –≤ —Ç–æ—á–∫–∞—Ö –æ–¥–Ω–æ–≥–æ –∏–∑ –º–∞—Å—Å–∏–≤–æ–≤, —Ä–∞–∑–º–µ—Ä—ã –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ ‚Äî 2*eps. –°–ª–æ–∂–Ω–æ—Å—Ç—å ‚Äî *O(N log N)*, –æ—á–µ–Ω—å —Ç—Ä—É–¥–æ–µ–º–æ–∫ –≤ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏.
3. –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –æ–¥–Ω–æ–≥–æ –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤ –ø–æ x (–∏–ª–∏ y)-–∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–µ, –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –¥–ª—è –∫–∞–∂–¥–æ–π —Ç–æ—á–∫–∏ –≤—Ç–æ—Ä–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –¥–æ–ø—É—Å—Ç–∏–º–æ–≥–æ x-–¥–∏–∞–ø–∞–∑–æ–Ω–∞ —á–µ—Ä–µ–∑ –±–∏–Ω–∞—Ä–Ω—ã–π –ø–æ–∏—Å–∫, –ø–µ—Ä–µ–±–æ—Ä –∫–∞–Ω–¥–∏–¥–∞—Ç–æ–≤. –°–ª–æ–∂–Ω–æ—Å—Ç—å ‚Äî *O(N log N)*, –Ω–æ –≤ —Å–ª—É—á–∞–µ –º–Ω–æ–∂–µ—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ —Å–æ–≤–ø–∞–¥–µ–Ω–∏—è x-–∫–æ–æ—Ä–¥–∏–Ω–∞—Ç –≤—ã—Ä–æ–∂–¥–∞–µ—Ç—Å—è –≤ *O(N¬≤)*

–ó–¥–µ—Å—å –¥–ª—è –ø—Ä–æ—Å—Ç–æ—Ç—ã ‚Äî —Å–ª–æ–∂–Ω–æ—Å—Ç–∏ –¥–ª—è –¥–≤—É—Ö –º–∞—Å—Å–∏–≤–æ–≤ –æ–¥–∏–Ω–∞–∫–æ–≤–æ–π –¥–ª–∏–Ω—ã.

–¢—Ä–µ—Ç–∏–π –≤–∞—Ä–∏–∞–Ω—Ç ‚Äî –∑–æ–ª–æ—Ç–∞—è —Å–µ—Ä–µ–¥–∏–Ω–∞:
1. –°–æ—Ä—Ç–∏—Ä—É–µ–º –æ–¥–∏–Ω –∏–∑ –º–∞—Å—Å–∏–≤–æ–≤ (–ø—É—Å—Ç—å A) –ø–æ x (–∏–ª–∏ y)-–∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–µ
2. –ó–∞—Ç–µ–º –¥–ª—è –∫–∞–∂–¥–æ–π —Ç–æ—á–∫–∏ P –∏–∑ B –Ω–∞—Ö–æ–¥–∏–º –Ω–∏–∂–Ω—é—é –∏ –≤–µ—Ä—Ö–Ω—é—é –≥—Ä–∞–Ω–∏—Ü—ã —á–µ—Ä–µ–∑
